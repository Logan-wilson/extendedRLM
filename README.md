# An extension of the Radial Line Model to predict spatial relations

This repository contains the official implementation of the Extended Radial Line Model proposed in our paper "**An extension of the Radial Line Model to predict spatial relations**", which has been accepted and published to the 18th International Joint Conference on Computer Vision, Imaging and Computer Graphics Theory and Applications, VISIGRAPP 2023 (Oral).

This project has been completed in 2022.


Paper available [here](https://www.scitepress.org/PublicationsDetail.aspx?ID=I1gqc7ZC9M8=&t=1).
